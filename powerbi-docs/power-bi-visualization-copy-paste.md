# <a name="copy-and-paste-a-visualization-in-power-bi"></a>Kopiowanie i wklejanie wizualizacji w usłudze Power BI
1. W usłudze Power BI otwórz raport w [widoku do edycji](service-reading-view-and-editing-view.md) i [utwórz wizualizację](power-bi-report-add-visualizations-i.md), jeśli raport jeszcze jej nie ma. 
2. Wybierz wizualizację i użyj klawiszy **Ctrl+C**, aby skopiować, oraz klawiszy **Ctrl+V**, aby wkleić.  
   ![](media/power-bi-visualization-copy-paste/copypasteviznew.gif)
